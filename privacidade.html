Política de Privacidade

A tua privacidade é uma prioridade para nós. Esta política explica de forma transparente como recolhemos, utilizamos e protegemos os teus dados pessoais.

1. Quem Somos

O AulaBot é uma plataforma educacional portuguesa que utiliza inteligência artificial para apoiar estudantes no seu percurso escolar. Estamos comprometidos em proteger a tua privacidade e em tratar os teus dados com total transparência e segurança.

Para qualquer questão relacionada com a proteção de dados, podes contactar-nos através de: privacidade@aulabot.pt

2. Dados que Recolhemos

Para te oferecer o melhor serviço possível, recolhemos os seguintes tipos de informação:

  • Dados de registo: Nome, endereço de email e palavra-passe encriptada quando crias uma conta
  • Dados de utilização: Perguntas enviadas ao AulaBot, histórico de conversas e preferências de disciplinas
  • Dados técnicos: Endereço IP (de forma anonimizada), tipo de navegador, sistema operativo e padrões de acesso
  • Imagens: Fotografias de exercícios ou apontamentos que envies para análise pelo sistema

Dados de menores: Se tens menos de 16 anos, deves obter autorização do teu encarregado de educação antes de utilizares o serviço.

3. Como Utilizamos os Teus Dados

Os teus dados são utilizados exclusivamente para:

  • Fornecer o serviço: Processar as tuas perguntas e gerar respostas personalizadas
  • Melhorar a experiência: Adaptar as respostas ao teu nível escolar e disciplinas selecionadas
  • Garantir a segurança: Prevenir fraudes, abusos e proteger o serviço de utilizações maliciosas
  • Comunicar contigo: Enviar notificações importantes sobre o serviço (podes optar por não as receber)
  • Aperfeiçoar o AulaBot: Analisar tendências de utilização para melhorar a qualidade das respostas

Importante: Nunca vendemos os teus dados a terceiros, nem os utilizamos para fins publicitários.

4. Base Legal para o Tratamento

Tratamos os teus dados pessoais com base nas seguintes fundamentos legais, em conformidade com o RGPD:

  • Execução de contrato: Para te fornecer o serviço que solicitaste ao criares uma conta
  • Consentimento: Para funcionalidades opcionais como notificações por email ou análise de imagens
  • Interesse legítimo: Para melhorar o serviço, garantir a segurança e prevenir fraudes
  • Obrigação legal: Quando a lei nos obriga a conservar ou partilhar determinadas informações

5. Partilha de Dados

Os teus dados pessoais só são partilhados nas seguintes circunstâncias:

  • Fornecedores de serviços: Empresas que nos ajudam a operar o AulaBot (ex: alojamento de servidores), sempre sob contratos de confidencialidade rigorosos
  • Obrigações legais: Quando exigido por tribunais, autoridades policiais ou reguladores
  • Com o teu consentimento: Apenas quando nos deres autorização explícita

Nunca partilhamos os teus dados para fins comerciais ou de marketing de terceiros.

6. Conservação dos Dados

Conservamos os teus dados apenas durante o tempo necessário:

  • Dados da conta: Enquanto mantiveres a conta ativa, e até 30 dias após eliminação
  • Histórico de conversas: Até 12 meses, para permitir continuidade nas sessões de estudo
  • Dados técnicos: Até 6 meses para fins de segurança e análise
  • Imagens enviadas: Processadas em tempo real e eliminadas automaticamente após 24 horas

7. Os Teus Direitos

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD), tens os seguintes direitos:

  • Direito de acesso: Podes solicitar uma cópia de todos os dados que temos sobre ti
  • Direito de retificação: Podes corrigir dados incorretos ou incompletos
  • Direito ao apagamento: Podes pedir que eliminemos os teus dados ("direito a ser esquecido")
  • Direito à portabilidade: Podes receber os teus dados num formato estruturado
  • Direito de oposição: Podes opor-te ao tratamento dos teus dados em certas circunstâncias
  • Direito de retirar consentimento: Podes retirar o consentimento a qualquer momento

Para exercer qualquer destes direitos, contacta-nos através de privacidade@aulabot.pt. Responderemos no prazo de 30 dias.

Se considerares que os teus direitos não foram respeitados, podes apresentar reclamação junto da Comissão Nacional de Proteção de Dados (CNPD) em www.cnpd.pt.

8. Segurança dos Dados

Implementamos medidas de segurança técnicas e organizativas para proteger os teus dados:

  • Encriptação de palavras-passe e dados sensíveis
  • Ligações seguras através de HTTPS
  • Acesso restrito aos dados apenas a pessoal autorizado
  • Monitorização contínua de acessos suspeitos
  • Backups regulares e seguros

Apesar de todos os esforços, nenhum sistema é 100% seguro. Recomendamos que utilizes uma palavra-passe forte e única para a tua conta.

9. Alterações a Esta Política

Podemos atualizar esta Política de Privacidade para refletir alterações nos nossos serviços ou requisitos legais. Se fizermos alterações significativas, notificar-te-emos através do email associado à tua conta.

Recomendamos que consultes esta página periodicamente para te manteres informado sobre como protegemos os teus dados.

10. Contacto

Para qualquer questão sobre privacidade ou proteção de dados, contacta-nos:

  • Email: privacidade@aulabot.pt
  • Suporte geral: suporte@aulabot.pt

Estamos empenhados em responder às tuas questões no prazo máximo de 5 dias úteis.
